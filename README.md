# Student Registration Portal

A clean, responsive, dark-themed Student Registration Form built using semantic HTML5 and custom CSS3 variables. 

## 🚀 Features
* **Comprehensive Data Fields**: Captures user credentials, department selections, skills tracking, date of birth, and profile image uploads.
* **Modern Dark Theme**: Features a distinct custom color palette leveraging CSS `:root` variables for consistent branding.
* **Responsive Typography**: Implements CSS Media Queries to adapt the heading layouts seamlessly from mobile screens to desktop displays.
* **Sticky Navigation**: A fixed header with depth shadows ensures persistent visibility during scrolling.
* **Form Optimization**: Standardized form inputs with autofocus, explicit field requirements, minimum length validation boundaries, and a clean reset mechanism.

## 🛠️ Tech Stack
* **Frontend**: HTML5, CSS3 (Custom Properties & Media Queries)
